# Marketing (Biblioteca de templates — Canva)

**Rota:** `/marketing` · categorias: `/marketing/estetica` · `/marketing/datas-comemorativas` · `/marketing/avisos`  ·  **Tipo:** Galeria de templates de design (integração Canva)  ·  **Screenshots:** `marketing.png`, `marketing-categoria.png`, `marketing-template-modal.png`

## Propósito

O módulo **Marketing** do Clínica Experts é uma **biblioteca de templates de design prontos para redes sociais**, **impulsionada pela Canva**. Não é uma ferramenta de campanhas/automação/disparo — é um **acervo de criativos** (posts e stories de Instagram) que o profissional edita no Canva e publica nas próprias redes.

> ⚠️ Importante: a expectativa de "campanhas / modelos de mensagem / automações / público" NÃO se aplica a este módulo. Esses recursos vivem em outras áreas (Comunicação / CliniChat para mensageria; "Avisos automáticos" e lembretes em Preferências/Agenda; CliniCRM no plano Experts). Marketing aqui = **design assets**.

## Layout — `/marketing`

- Banner roxo/gradiente: **"O que você vai criar hoje?"** + **busca de conteúdo** ("Pesquisar conteúdo") · selo **Impulsionado pela Canva**.
- Carrosséis horizontais por **categoria**, cada um com link **Ver tudo**:
  - **Estética** → `/marketing/estetica`
  - **Datas comemorativas** → `/marketing/datas-comemorativas`
  - **Avisos** → `/marketing/avisos`
- Card especial **"Criar no Canva"** (abre editor Canva em branco).
- Tabs/breadcrumb no topo: Marketing › {categoria}.

## Página de categoria (ex.: `/marketing/estetica`)

- Banner com nome da categoria + busca.
- **Sub-seções temáticas** (carrosséis), ex. em Estética: **Facial · Corporal · Podologia · Bem-estar · Melasma · Nutricosméticos** — taxonomia rica por tema clínico/estético. Cada sub-seção tem "Ver tudo".

## Card de template

- Miniatura do design + **título** (ex.: "Instagram Post Nutricosméticos Laranja", "Instagram Post Alimentação e Acne Elegante Laranja").
- **Tags** (ex.: estética, laranja, corporal, cosméticos / saúde, acne).
- **Formato/dimensão**: "Post • 1080 px × 1080" (também Stories).
- Ação **"Editar no Canva"** → abre o template no editor Canva (externo) para personalização. (NÃO acionado — abre serviço externo.)

## Fluxo
- Buscar/navegar template → "Editar no Canva" → personaliza no Canva → baixa/publica nas redes. Sem disparo dentro do app.

## Observações para o Artemise

- **Marketing = parceria de conteúdo (Canva), não engine de campanha.** Decisão de produto: em vez de construir editor de design próprio, integraram a Canva e curaram uma biblioteca de templates segmentada por nicho de clínica (estética facial/corporal, podologia, datas comemorativas, avisos). Baixo custo de construção, alto valor percebido para o profissional que não tem designer.
- A **curadoria por tema clínico** (Melasma, Acne, Nutricosméticos, Podologia) é o diferencial — não é Canva genérico, são criativos prontos para o vocabulário do nicho.
- Para o Artemise: se o objetivo é marketing, o caminho barato é o mesmo (integração Canva + biblioteca curada). O caminho de maior valor (e que o Clínica Experts NÃO faz aqui) seria **campanhas reais**: segmentar público por etiquetas/CRM e disparar por WhatsApp/E-mail/SMS — isso eles empurram para CliniCRM/Comunicação (planos altos). Há espaço para diferenciar unificando criativo + disparo + público num só módulo.
- Datas comemorativas como categoria = gancho de engajamento sazonal pronto (reduz fricção de "o que postar hoje").
